# FitTrend-Store
FitTrend Store è un e-commerce dedicato agli appassionati di palestra, fitness e allenamento in casa. Il sito permette di acquistare articoli utili per l’attività fisica, come accessori da palestra, piccoli attrezzi, abbigliamento sportivo e prodotti molto richiesti sui social, ad esempio shaker, guanti, fasce, tappetini e manubri regolabili.
